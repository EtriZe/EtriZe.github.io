Résumé

Jeu de plateforme 2D infini développé avec Unity et C# pour apprendre les bases du moteur. Le joueur survit le plus longtemps possible en sautant sur des plateformes qui défilent vers la gauche. Infinite 2D platformer developed with Unity and C# to learn the engine's basics. The player survives as long as possible by jumping on platforms scrolling to the left.

Ce que j'ai développé

  • Génération procédurale de plateformes : spawn aléatoire, défilement automatique et destruction hors champ, avec un nombre constant de plateformes à l'écran. Procedural platform generation: random spawn, automatic scrolling and off-screen destruction, with a constant number of platforms on screen.
  • Double saut avec détection directionnelle pour distinguer un atterrissage d'un simple contact latéral. Double jump with directional detection to distinguish a landing from a simple lateral contact.
  • Score incrémenté à chaque atterrissage, avec sauvegarde du meilleur score entre les parties. Score incremented on each landing, with best score saving between sessions.
  • Navigation entre scènes : menu principal, jeu et écran de game over. Scene navigation: main menu, game and game over screen.

Ce que j'ai appris

Une première prise en main d'Unity : cycle de vie des objets, communication entre scripts, physique 2D et architecture multi-scènes. A first hands-on experience with Unity: object lifecycle, inter-script communication, 2D physics and multi-scene architecture.
